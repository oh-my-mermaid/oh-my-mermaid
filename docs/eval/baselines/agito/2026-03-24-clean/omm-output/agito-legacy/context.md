The legacy codebase is agito v0.4.8 — the previous Bun monorepo before the v2 rewrite. It is kept in the repo as a reference boundary. The MCP server packages (session-mcp-server, bridge-mcp-server) represent an earlier architectural approach where agent sessions were mediated via MCP protocol rather than direct PTY spawning. The v2 agito-app dropped this in favor of direct node-pty engine management. Some packages (shared, codex-types) may still contain reusable type definitions being migrated to v2.

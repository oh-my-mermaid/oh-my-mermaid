The desktop app is the primary user-facing product. It is distributed as a macOS/Windows binary via electron-builder. Users install it locally and bring their own Claude Code CLI credentials — Agito spawns it via node-pty rather than calling Anthropic APIs directly. The app connects to @agito-server for cloud-backed image generation and Supabase for auth/billing state. agito-legacy was the previous monorepo version; agito-app is a clean rewrite with tighter scope.

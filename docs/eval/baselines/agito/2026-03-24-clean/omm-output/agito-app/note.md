The desktop app intentionally does NOT use the Anthropic Agent SDK query() function. It spawns the user's own Claude Code CLI via node-pty — the same relationship as iTerm2 or VS Code Terminal running Claude Code. This is a deliberate ToS compliance choice documented in chat-implementation-architecture.md. The Phase 2 plan (assistant-ui rich chat UI) is not yet implemented; Phase 1 is the xterm.js terminal panel. Internal subsystems (character engine, office world rendering, IPC bridge) are intentionally kept as internal modules within this package, not promoted to separate top-level classes.

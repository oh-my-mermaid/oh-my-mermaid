Electron + Vite + TypeScript desktop application — the primary product of agito. Renders a 2D pixel-art office world (PixiJS) where AI agent "characters" are persistent avatars bound to engine sessions (Claude Code, Codex) via node-pty PTY spawning. Three process layers: main (Node.js), preload (IPC bridge), renderer (React + PixiJS). Manages local character state, terminal sessions, and office layout via Zustand stores and sql.js.

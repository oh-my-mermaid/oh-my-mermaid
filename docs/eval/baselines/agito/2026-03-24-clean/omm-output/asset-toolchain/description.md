Root-level Python scripts for batch generation of game assets (sprites, minerals, lockscreens) used in the agito office world. Scripts call the agito-server Gemini image generation pipeline directly. Includes: gito_cyan_chroma.py (cyan crystal sprites with chroma-key green removal), gito_mineral_gen.py (4x4 mineral sprite sheet generation and splitting), gito_magic_crystal_gen.py (magic crystal variants), gito_split_careful.py and gito_split_floodfill.py (sprite sheet splitting utilities), lockscreen_gen.py (phone lockscreen wallpaper generation).

These scripts are developer/content-creator tooling — not deployed services. They are run locally by developers to batch-generate game assets that are then committed to agito-app/builtin-assets/. They use the agito-server Gemini pipeline and/or direct Gemini API calls with Pillow for post-processing (background removal, chroma keying, sprite sheet splitting). The scripts are standalone Python files at the repo root; they require the same Python environment as agito-server (or a compatible subset).

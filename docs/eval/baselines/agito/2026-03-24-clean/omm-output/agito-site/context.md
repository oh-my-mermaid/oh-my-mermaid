The site is the public web presence and auth entry point for agito users. It handles OAuth flows via Supabase (the /auth routes) and billing redirects via Polar. The /dev routes appear to be internal tooling pages (not publicly linked). The site and desktop app share the same Supabase project for auth state — a user logged in on the site has the same identity in the desktop app. Uses pnpm workspace (pnpm-workspace.yaml present) but operates as a standalone Next.js deployment.

The server is deployed as a Docker container (Dockerfile present). It is the cloud backend for the desktop app — handling image/sprite generation that is too compute-intensive or API-key-sensitive for the client. Alembic manages DB migrations. The server does NOT orchestrate AI agent sessions; those run entirely inside the Electron main process via node-pty. The root-level gito_*.py scripts call this server's generation endpoints directly for batch asset creation workflows.

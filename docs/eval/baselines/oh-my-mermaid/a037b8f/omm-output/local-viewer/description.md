The local web viewer for browsing architecture diagrams in a browser. Starts an HTTP server (`omm view`) that serves a single-page HTML viewer with live-reload via Server-Sent Events (SSE). Exposes a JSON REST API (`/api/classes`, `/api/class/:name`, `/api/class/:name/diff`, `/api/class/:name/refs`) backed by the store and doc-analysis subsystems. Watches the `.omm/` directory for file changes and pushes update events to all connected SSE clients.

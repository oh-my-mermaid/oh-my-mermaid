The viewer is a pure Node.js HTTP server with no framework dependencies — built on `node:http` directly. The SPA (`viewer.html`) is a static file bundled into the dist. The server injects the project name into the HTML at serve time via string replacement. File watching (`src/server/watcher.ts`) uses `fs.watch` or similar to detect changes under `.omm/` and broadcasts to all active SSE connections. Port collision is handled gracefully: if the requested port is in use the server tries `port + 1` automatically. The API layer (`src/server/api.ts`) is a simple pattern-match router (regex on pathname) with no routing framework. All API responses include `Access-Control-Allow-Origin: *` for local dev convenience.

Detects installed AI coding tools and registers omm skills with them. Supports Claude Code, Cursor, Codex (OpenAI), OpenClaw, and Antigravity. Each platform has a driver (`src/lib/platforms/<name>.ts`) that implements `detect()`, `isSetup()`, `setup()`, and `teardown()`. The `omm setup` command auto-detects all installed platforms and installs the skill manifests; it can also target a specific platform or perform a full teardown. Skills are sourced from the `skills/` directory bundled with the npm package.

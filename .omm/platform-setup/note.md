The platform list (`ALL_PLATFORMS`) in `index.ts` is the single source of truth for supported tools. Adding support for a new AI coding tool only requires adding a new driver file and registering it there. The version-matching in `isSetup()` acts as an auto-update prompt: after `omm update`, all platforms will report as not-setup until `omm setup` is re-run. This ensures skills stay synchronized with the CLI version.
